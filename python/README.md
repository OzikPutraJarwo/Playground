# Python README and How-to

## 1. Creating a virtual environment

Go to project's directory:

```
cd projects-name
```
Create a venv:

```
python3 -m venv venv
```

Activate the venv:

```
source venv/bin/activate
```

## 2. Installing dependencies

Create a venv, then run:

```
pip install -r requirements.txt
```
